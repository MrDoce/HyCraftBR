# HyCraft BR

Site informativo do servidor HyCraft BR com chat em tempo real.

## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub
2. Execute no terminal:
```bash
git init
git add .
git commit -m "Primeira versão"
git remote add origin https://github.com/SEU_USUARIO/hycraft-br.git
git branch -M main
git push -u origin main
```
3. Vá em Settings > Pages e ative com branch `main` e pasta `/ (root)`
4. Configure o Firebase no `script.js` para o chat funcionar

## Estrutura

- `index.html` - Página principal
- `styles.css` - Estilos
- `script.js` - Funcionalidades e chat
